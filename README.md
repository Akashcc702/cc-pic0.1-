# CC Pic Bot 🤖🎨

Telegram bot for AI image generation using HuggingFace Inference API.

## Models Supported
- FLUX.1-schnell (black-forest-labs)
- Stable Diffusion XL (stabilityai)
- SD 3.5 Large Turbo (stabilityai)
- SDXL Lightning (ByteDance)

## Commands
- `/start` - Welcome message
- `/generate <prompt>` - Generate image
- `/model` - Choose AI model
- `/models` - List models
- `/help` - Help guide
